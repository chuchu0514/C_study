void push(element item) {
    // 꽉 찼으면 doubling
    if (top >= capacity - 1) {
        capacity *= 2;
        stack = (element*)realloc(stack, capacity * sizeof(element));
    }
    stack[++top] = item;
}

ex)
push 1번 → capacity 1→2 doubling → 1개 copy
push 2번 → capacity 2→4 doubling → 2개 copy
push 3번 → 여유있음
push 4번 → capacity 4→8 doubling → 4개 copy
push 5번 → 여유있음

push가 N번일 때 doubling 횟수 = log₂N
push 17이면 doubling횟수 5 소숫점 있으면 무조건 올림 
doubling이 k번이면 total copy 횟수는 {2(k승) - 1} ->copy에 대한 시간복잡도는 그래서 O(2(k승)
근데 push가 n번일 때 dougling은 log₂N 번이니까 k = log₂N  즉 O(N)
